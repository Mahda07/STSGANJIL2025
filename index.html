JAWABAN

1.Create, Read, Update, dan Delete (CRUD) adalah kerangka kerja fundamental untuk manajemen data dalam aplikasi. Keempat operasi dasar ini mencakup semua proses manipulasi data dalam database, menjadikannya elemen esensial bagi developer.

2. - Collection: Kumpulan dokumen yang terkait dan disimpan dalam basis data Firestore. - addDoc: Menambahkan dokumen baru ke dalam collection dengan ID dokumen yang unik. - deleteDoc: Menghapus dokumen dari collection berdasarkan ID dokumen yang ditentukan.

3.asynchronous dalam javaScript adala teknik pemograman yang memungkinkan kode berjalan secara pararel, tanpa menggunkan proses sebelum selesai. ini memungkinkan kode untuk berjalan lebih cepat dan efisien.

4.Berikut adalah langkah-langkah dalam membuat program menggunakan SPCK, GitHub, dan Firebase:

Langkah 1: Membuat Proyek di GitHub 1. Buat akun GitHub jika belum memiliki. 2. Buat repository baru untuk proyek Anda. 3. Tentukan nama repository dan deskripsinya.

Langkah 2: Membuat Proyek di Firebase 1. Buat akun Firebase jika belum memiliki. 2. Buat proyek baru di Firebase. 3. Tentukan nama proyek dan negara Anda. 4. Aktifkan fitur-fitur yang dibutuhkan (misalnya, Firestore, Authentication).

Langkah 3: Menginstal SPCK (Flutter) 1. Unduh dan instal Flutter di komputer Anda. 2. Instal IDE (Integrated Development Environment) seperti Android Studio atau Visual Studio Code. 3. Buat proyek baru di IDE Anda.

Langkah 4: Menghubungkan Proyek dengan Firebase 1. Tambahkan Firebase SDK ke proyek Anda. 2. Konfigurasi Firebase di proyek Anda. 3. Uji coba koneksi dengan Firebase.

Langkah 5: Membuat Aplikasi 1. Buat desain antarmuka pengguna (UI) untuk aplikasi Anda. 2. Tambahkan fitur-fitur yang dibutuhkan (misalnya, autentikasi, penyimpanan data). 3. Uji coba aplikasi Anda.

Langkah 6: Mengunggah Aplikasi ke GitHub 1. Buat commit untuk perubahan-perubahan di proyek Anda. 2. Unggah proyek Anda ke GitHub. 3. Pastikan proyek Anda dapat diakses secara publik.

Langkah 7: Membuat Aplikasi Siap Rilis 1. Buat aplikasi Anda siap rilis dengan mengatur konfigurasi rilis. 2. Unggah aplikasi Anda ke toko aplikasi (misalnya, Google Play Store). 3. Pastikan aplikasi Anda dapat diunduh dan diinstal dengan benar.
